# Scoop bucket for owa-bridge

Install the local-first Outlook Web CLI and MCP server:

```console
scoop bucket add owa-bridge https://github.com/nkiyohara/scoop-owa-bridge
scoop install owa-bridge/owa-bridge
```

The manifest installs checksummed Windows archives published by
[owa-bridge](https://github.com/nkiyohara/owa-bridge). It does not require a
Microsoft Graph application or a hosted bridge.
